# WebDav

### WebDAV config

| Name     | Value                      |
| -------- | -------------------------- |
| Url      | http[s]://domain:port/dav/ |
| Host     | domain                     |
| Path     | dav                        |
| Scheme   | http/https                 |
| Port     | Same as web port           |
| Username | See manage                 |
| Password | See manage                 |
