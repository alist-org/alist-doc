# WebDav

### WebDAV 配置

| 参数 | 值                         |
| ---- | -------------------------- |
| 链接 | http[s]://domain:port/dav/ |
| 主机 | domain:port                |
| 路径 | dav                        |
| 协议 | http/https                 |
| 端口 | 同 web 端口                |
| 账号 | 见后台                     |
| 密码 | 见后台                     |
